WOOSC - Wickwar Out of School Club

For this website, I was asked to make a fun, colour full, child-friendly website. I decided that pastel colours would suit this theme nicely and having separators between sections would add to the fun feeling of the website. WOOSC also asked for the website to have gallery pages that suit images of any size and a 'meet the team' page that can have a different number of employees over time.

KLB School Intranet

For this website I was asked to update an old intranet page into a more modern version. This was intended to be used for searching, so included a Google search bar as well as some quick links to commonly used webpages. I also made it so the background would refresh each time the page was loaded.
